🎯 LIMS - Herramientas de Asignación Masiva

Versión: 1.0.0

Autor: Omar Martinez

Compatibilidad: Odoo 18, requiere lims_analysis_v2

📋 Descripción

Módulo de productividad que extiende las funcionalidades de lims_analysis_v2 con herramientas de asignación masiva para reducir tareas repetitivas y mejorar la eficiencia del laboratorio.

🎯 Características Principales

👤 Asignación Masiva de Analistas

  • Asignar analista responsable a múltiples parámetros simultáneamente
  • Verificación PIN para seguridad y trazabilidad
  • Opción de sobrescribir asignaciones existentes

🧪 Asignación Masiva de Medios de Cultivo

  • Usar sets de medios predefinidos para asignación rápida
  • Seleccionar medios individuales para múltiples parámetros
  • Copiar configuración de medios desde otro parámetro
  • Configuración automática de incubación

📅 Asignación Masiva de Fechas

  • Fechas generales de análisis (inicio, compromiso)
  • Fechas específicas por tipo de proceso
  • Inclusión de horas de procesamiento

🔧 Asignación Masiva de Equipos

  • Equipos de procesamiento por tipo de proceso
  • Equipos de incubación para medios específicos
  • Equipos adicionales con uso personalizado

📊 Cambio Masivo de Estados

  • Estados de análisis (sin procesar, en proceso, finalizado)
  • Estados de reporte (borrador, listo)
  • Configuración de procesos requeridos
  • Filtros avanzados por estado actual o tipo de resultado

🏭 Asignación Masiva de Ambientes

  • Ambientes de procesamiento por tipo de proceso
  • Asignación de equipos específicos (campanas)
  • Filtros por tipo de resultado (cualitativo/cuantitativo)

⚡ Beneficios

🚀 Eficiencia: Reduce significativamente el tiempo de captura de datos repetitivos
✅ Consistencia: Minimiza errores por asignaciones manuales repetitivas
🎯 Productividad: Permite a los analistas enfocarse en el trabajo científico
🔒 Trazabilidad: Mantiene todos los controles de seguridad del sistema base

🚀 Instalación

  1. Asegurar que lims_analysis_v2 esté instalado
  2. Copiar este módulo a la carpeta addons
  3. Actualizar lista de módulos
  4. Instalar lims_analysis_mass_tools
⚠️ Importante: Este módulo requiere lims_analysis_v2 y lims_analysis_config para funcionar correctamente.

📖 Uso

Desde Lista de Parámetros

  1. Navegar a cualquier lista de parámetros de análisis
  2. Seleccionar múltiples parámetros
  3. Usar botón "🎯 Asignación Masiva" del menú Acción
  4. Seleccionar tipo de asignación deseada

Desde Formulario de Análisis

  1. Abrir cualquier análisis de muestra
  2. En la pestaña "Parámetros de Análisis"
  3. Usar botones de "Herramientas de Productividad"
  4. Los cambios se aplicarán a todos los parámetros del análisis
